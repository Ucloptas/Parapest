## рҹҺ® What is this?

> **30-second pitch:** Parents type chores into a dashboard. Kids see those chores as **little animals scattered around a parallax-scrolling forest**. Walk up, press `E`, "I Did It!", parent approves, points appear, kid trades points for rewards. That's it. That's the game.

```mermaid
sequenceDiagram

    G-->>K: popup: "I Did It!"
    K->>G: click

    A-->>G: { pending: true, points: 10 }

    P->>A: GET /api/pending-completions
    A-->>P: list (with badge count)
    P->>A: POST /api/pending-completions/:id/approve
    A->>A: +10 pts to child<br/>insert completedChores

    Note over K: next HUD tick<br/>shows new points
```

## рҹҡҖ Quick start (~5 min)

### Requirements

| Tool | Version | Why |
|:---|:---:|:---|
| [Godot](https://godotengine.org/download) | **4.5** | Game client |

| npm | latest | Install deps |

### 1пёҸвғЈ Backend

```powershell
cd "Sprint 2 backend"
npm install
npm start
```

You should see something like:

```
Database initialized
Server running on port 49152
Using SQLite database: вҖҰ/database.db
Port written to вҖҰ/server_port.txt
```

> рҹӘҹ Windows install drama (sqlite3 build errors etc.)? Try `Sprint 2 backend/INSTALL.md`, `QUICK_FIX.md`, `FIX_SDK.md`, or `SETUP_WINDOWS.md`.

### 2пёҸвғЈ Godot client

1. Open this repo folder in **Godot 4.5**
2. Press **F5** (Play)

> рҹ’Ў **Pro tip:** `HTTPClient.gd` will **auto-launch the backend** for you on game start (`npm run dev`). If it's already running on a different port, the game reads `server_port.txt` to find it. So in practice you can usually skip step 1пёҸвғЈ.

|:---|:---|

## рҹҗӣ Something broke?

| Symptom | Try |
|:---|:---|
| рҹҳ¶ Can't login / "Connection error" | Is the backend up? Check Godot's **Output** panel for `HTTPClient:` lines. |
| рҹ”ў Wrong port | `server_port.txt` is the truth; Godot reads it on startup. Restart the game so it re-reads. |
| рҹ’Ҙ `npm install` fails on Windows (sqlite3) | [`INSTALL.md`](Sprint%202%20backend/INSTALL.md), [`QUICK_FIX.md`](Sprint%202%20backend/QUICK_FIX.md), [`FIX_SDK.md`](Sprint%202%20backend/FIX_SDK.md), [`SETUP_WINDOWS.md`](Sprint%202%20backend/SETUP_WINDOWS.md) |
| рҹ§Ҡ Godot won't open project | Use **Godot 4.5** specifically вҖ” not 3.x, not earlier 4.x betas |
| рҹҰ— Kid says "I did it!" but no points | That's the new flow вҖ” go check the **Approvals** tab on the parent dashboard |
| рҹ‘» Animals didn't spawn | Parent has to **add chores/rewards first** вҖ” animals = chores/rewards |
| рҹӘҹ Backend window won't die | `taskkill /F /IM node.exe` *(nuclear, but works)* |
